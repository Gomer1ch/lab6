Завдання 1:
Створити меню MenuStrip з командами Input color, Change, Exit, Help.
При виборі команди Exit додаток завершує роботу. При виборі команди 
Input color відкривається діалогове вікно, що містить:
• три поля введення типу TextBox з мітками Red, Green, Blue;
• групу з двох прапорців (Left, Right) типу CheckBox;
• кнопку типу Button.
При виборі команди Change колір головного вікна змінюється на заданий 
(ліва, права або обидві половини вікна в залежності від установки 
прапорців).
Завдання 2:
Додайте на формі дві кнопки (Button) і одну мітку (Label).
Зробіть на кнопках наступні написи: «Скрыть», «Показать». Створіть 
обробники події натискання на дані кнопки, які будуть приховувати або 
показувати мітку. Створіть обробник події створення форми (Load), який 
буде встановлювати колір форми і змінювати текст мітки на рядок «Начало 
работы».
Завдання 3: 
Створити програму, що додає різні елементи управління на форму і на 
панель Panel. Тип елементів управління вибирається випадковим чином. 
Передбачити можливість виведення інформації про кількість елементів за 
типами та інформацію про розташування елементів.
